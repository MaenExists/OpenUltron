# Web Research Skill 🌐

## Goal

Find **credible, up‑to‑date** information, capture the evidence, and record what remains unknown.

## Research Workflow

- Define the question in one sentence
- Identify time sensitivity and required freshness
- Run a broad `search_web` query
- Scan results for authoritative sources
- Use `fetch_url` on the top sources
- Extract the key facts and store them in memory
- Record uncertainties and next follow‑ups

## Source Quality Checklist

- Prefer official documentation, reputable news, or primary sources
- Cross‑check claims across multiple sources
- Avoid summaries when original sources are available
- Note publication date and context

## Query Patterns

- Use exact phrases with quotes for names or API terms
- Add a year or “latest” for changing information
- Include the provider name to reduce ambiguity

## Evidence Log Template

```markdown
# Research Log

## Question
What is the current OpenAI Python SDK response format for tool calling?

## Sources Consulted
- URL 1 (date)
- URL 2 (date)

## Key Findings
- Finding A
- Finding B

## Open Questions
- What is the provider‑specific deviation, if any?
```

## Exit Criteria

- At least two reliable sources agree
- The answer is current for the needed timeframe
- Unknowns are listed if the evidence is incomplete
